Packages Needed
---
- MySQL
- Freeradius3
- Coova-Chilli `(For Loginpage)`
- php `7` or `8`
- php-mod-mysqli, php-mod-pdo-mysql, php-mod-curl, php-mod-gd

Features
---
- Add, Edit & Remove Plans
- Add, Edit & Remove Bandwidth
- Add & Remove Users
- Add & Remove Batch
- Mac Binding Support
